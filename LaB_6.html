📚 Теоретическая справка

Set

  • Коллекция уникальных значений
  • Автоматически удаляет дубликаты
  • Методы: add, delete, has, clear
  • Размер: size

Map

  • Коллекция ключ-значение
  • Ключ может быть любого типа
  • Методы: set, get, delete, has
  • Итерация в порядке вставки
